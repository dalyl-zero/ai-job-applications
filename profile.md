# Candidate Profile Template

Use this file to store the factual source material for resume and cover letter generation.
Keep it concise, specific, and up to date.

## Background

- Current location and work authorization (only what is relevant for hiring).
- Education history (degree, field, institution, dates).
- Total years of experience.
- Preferred role types (for example: full-time, internship, working student, contract).

## Language Proficiency

- Native and professional languages with level (for example: C1, B2).

## Technical Skills & Strengths

- Frontend stack.
- Backend stack.
- Databases.
- Tooling and delivery practices (for example: CI/CD, testing, Docker, cloud).
- Architecture strengths and engineering focus areas.

## Professional Experience Summary

- For each relevant role, include:
	- Company and location.
	- Job title.
	- Dates.
	- Main responsibilities.
	- Measurable outcomes.
	- Technologies used.

## Selected Projects

- Include 2-5 projects that best support target roles.
- For each project, include:
	- Project name and context.
	- Your ownership.
	- Scope/scale.
	- Outcome/impact.
	- Link (if public and relevant).

## Positioning

- What differentiates you for your target roles.
- What strengths should be emphasized in applications.
- Any context that should be handled carefully in wording.

## Constraints

- Target countries/regions and language requirements.
- Role or industry preferences.
- Non-negotiables (for example: visa constraints, remote/hybrid requirements, start date).

## Writing Rules

- Do not include sensitive personal data beyond what is needed for applications.
- Do not fabricate roles, responsibilities, or technologies.
- Prefer measurable statements over generic descriptors.
